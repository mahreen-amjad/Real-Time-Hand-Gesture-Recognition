# Real-Time Hand Gesture Recognition using YOLO

A real-time hand gesture recognition system trained on custom datasets using YOLO and OpenCV in Python.

## Features
- Real-time hand gesture tracking via webcam feed.
- Custom-trained YOLO model optimized for low latency and high accuracy.
- Dynamic on-screen bounding boxes and text labels for detected gestures.

## Tech Stack
- **Language:** Python
- **Frameworks:** YOLO (Ultralytics), PyTorch
- **Libraries:** OpenCV, NumPy
- **Editor:** Visual Studio Code (VS Code)
